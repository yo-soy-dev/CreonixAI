## ‚óà Spin It Up

### Prerequisites

Node.js v18+ and accounts for:
[Neon](https://neon.tech) ¬∑ [Clerk](https://clerk.com) ¬∑ [Google AI Studio](https://aistudio.google.com) ¬∑ [ClipDrop](https://clipdrop.co) ¬∑ [Cloudinary](https://cloudinary.com) ¬∑ [Brevo](https://brevo.com)

### Steps

**1 ‚Äî Clone**
```bash
git clone https://github.com/yo-soy-dev/AI-Saas-App.git
cd AI-Saas-App
```

**2 ‚Äî Install**
```bash
# Backend
npm install

# Frontend
cd client && npm install && cd ..
```

**3 ‚Äî Environment**
```bash
cp server/.env.example server/.env
# Fill in your keys ‚Äî see Environment Variables below
```

**4 ‚Äî Run**
```bash
# Terminal 1 ‚Äî Backend
cd server && npm run server     # ‚Üí http://localhost:5000

# Terminal 2 ‚Äî Frontend
cd client && npm start          # ‚Üí http://localhost:3000
```
